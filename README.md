----
本課程可以直接使用 colab.research.google.com 進行練習，或
在本機執行：請安裝以下軟體：
- Visual Studio Code
- Anaconda 

執行虛擬環境：
- conda create --name myvenv python=3.10
- conda activate myvenv

在 Python 虛擬環境中，請安裝以下模組：
- requests
- bs4
- jieba
- numpy 
- jupyterlab
- pillow
- matplotlib
- wordcloud 

安裝完畢之後，即可使用 jupyter lab 進入程式執行環境
